# 触发式技能

一个简单的触发式技能示例，支持自定义触发词和回应。

## 功能

这个技能提供了简单的命令响应功能：

- **小灯胜算** - 当说出触发词"小灯胜算"时，给予特定回应
- **保存** - 模拟保存动作，回应用户请求

## 使用方法

在对话中输入以下命令即可触发相应回应：

```
小灯胜算
保存
```

## 文件结构

```
test/
├── SKILL.md              # 技能定义文件（OpenClaw 必需）
└── references/
    ├── main.py           # 主处理函数
    └── skill.json        # 技能配置（可选）
```
