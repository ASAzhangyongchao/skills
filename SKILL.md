# 金灯塔胜算 Skill

## 描述

本 skill 集成飞书授权体系与胜算业务系统，支持用户通过自然语言查询 BI 数据报表。
Claude 自动识别用户意图，调用对应系统 API 获取数据，生成标准化 HTML 可视化报表，
上传至 OSS 并返回预览链接及数据总结建议，最终可将报表发布至胜算平台或通过定时任务自动推送。

---

## 触发条件

以下场景 Claude 应主动激活本 skill：

- 用户提及"报表"、"数据看板"、"BI"、"统计"、"分析"等关键词
- 用户询问某系统的业务数据（如销售额、订单量、用户数等）
- 用户要求"生成图表"、"导出数据"、"发布报表"、"定时推送"

---

## 核心能力

1. **飞书授权集成** - 一键获取用户手机号、消息发送、定时任务权限
2. **多系统支持** - 动态加载胜算平台支持的业务系统列表
3. **智能意图识别** - 自然语言解析，自动匹配 API 接口
4. **HTML 报表生成** - ECharts 可视化 + 数据表格 + 智能总结建议
5. **OSS 自动上传** - 生成即上传，返回永久预览链接
6. **定时任务推送** - 支持每日/每周/Cron 表达式定时生成推送
7. **胜算平台发布** - 一键发布到胜算报表中心

---

## 快速开始

```

```

首次使用会引导完成：
1. 飞书授权确认
2. 选择目标业务系统
3. 加载系统 API 文档

完成后即可直接说：
- "给我看上个月各渠道的销售额趋势"

## 错误处理规范

| 场景 | 处理方式 |
|------|----------|
| 飞书授权拒绝 | 明确告知缺少哪项权限，阻塞流程，引导手动开启 |
| 手机号获取失败 | 提示确认飞书账号绑定状态 |
| 无系统权限 | 明确告知缺少哪个系统权限，建议联系管理员 |
| Token 过期 | 自动静默刷新，对用户透明 |
| API 参数缺失 | 逐一追问用户，不静默填充默认值 |
| API 返回数据为空 | 告知用户，询问是否调整筛选条件 |
| API 调用失败 | 展示错误码与描述，不继续生成报表 |
| OSS 上传失败 | 提示失败原因，提供重试，不输出无效链接 |
| 定时任务执行失败 | 通过飞书消息通知用户，附错误说明 |
| 发布接口失败 | 展示失败原因，不影响已生成的 OSS 预览链接 |
